Frequently Asked Questions

How do I install a theme code?

  1. Click the code link for the theme
  2. Press Ctrl+A to select all
  3. Press Ctrl+C to save it to the clipboard
  4. Go to your blog and click the palette button on the top right to launch the theme editor
  5. Click the Edit HTML link (top left)
  6. Press Ctrl+A to select all
  7. Press the Backspace key to remove the existing code
  8. Press Ctrl+V to paste the code you copied earlier
  9. Click Update Preview and Save

*If you are using a Mac, use the Command key instead of Ctrl

I found a bug?

  • Make sure you have the latest version of the theme.
  • Remove any custom elements like music players or cursors.
  • Try to verify that the problem is with the theme and not with Tumblr itself.

After completing these three steps, send an email to atthemehelp@outlook.com. This email is for bugs only; any requests for features or customizations will be ignored. We will not help you edit your theme code.

Why are all of my new photo/video/etc posts showing up as text posts?

Tumblr made a change in the mobile app that causes posts made from the app to always be treated as text posts. If you want your posts to display normally, you have to post them from desktop. Any posts you have already made will need to be deleted and reposted.

Why do my photosets look messed up when posted from the app?

Update your theme code

How do I change posts per page?

Go to your edit theme page. Scroll down to the bottom of the settings panel and click Advanced options. The setting will be listed here.

When I go to my blog on my phone, my theme doesn’t show up?

Go to your edit theme page. Scroll down to the bottom of the settings panel and click Advanced options. Make sure that Use default mobile theme and Enable Google AMP are both unchecked. Note that the Tumblr app doesn’t show custom themes, you have to look at it from Chrome/Safari

Can I move or delete the theme credit?

Do whatever you want. Just don’t claim the theme as your own work.
