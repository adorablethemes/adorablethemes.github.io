Update history

3.2 (August 27, 2019)

  • Add a “load more posts” button, so that when there aren’t enough posts to make the window scrollable, you can still load the next page of posts
  • Fix infinite loading bug when a post has a broken image

3.1.1 (December 1, 2018)

  • More npf fixes

3.1 (October 6, 2018)

  • Support npf_row
  • Support video tag embeds

3.0 (September 13, 2018)

  • Dashboard-style reblog comments
  • Support mobile app text colors
  • Fix photosets in Mobile Safari
  • Brought over various minor tweaks from Copycat
  • Post links now use post text color instead of accent color
  • Faster page loads with ‘collapse photosets’ enabled as the unnecessary images are no longer loaded
  • Add alternate styles for the nav bar

2.1.1 (April 9, 2018)

  • Fix large empty space under Tumblr videos

2.1 (January 31, 2018)

  • Lots of behind-the-scenes work to stop posts from loading in overlapped
  • Added zoom button for photos and photosets
  • Post borders and shadows can now be toggled
  • Nav bar icons can now be toggled
  • Added Pin button on permalink pages
  • Redesigned post hover appearance
  • Various visual tweaks

2.0.5 (April 7, 2016)

  • Fix Disqus integration when HTTPS is enabled

2.0.4 (August 21, 2015)

  • Code for image sizing rewritten to fix any remaining bugs

2.0.3 (March 29, 2015)

  • Fix for some images being cropped

2.0.2 (March 20, 2015)

  • Workaround for a Tumblr bug breaking permalink pagination blocks

2.0.1 (March 1, 2015)

  • Fixed images in posts made after Feb 2015 being stretched

2.0 (July 7, 2014)

  • Global appearance options support
  • Responsive design
  • Better post hovers
  • Many more customization options
